Aplicação em Delphi 7 da Loja MFB MATERIAIS ELETRICOS S/A.

Funcionalidades da aplicação.

1- Cadastro de Fornecedores.
2- Cadastro de Produtos.
3- Cadastro de Clientes.
4- Tela de Vendas.
5- Relatórios.

IMPORTANTE:
O Sistema roda em Banco de Dados Mysql 5.7 via ODBC usando Driver ANSI.
Antes de executar a aplicação é necessario ter o Banco de Dados instaladoe driver ODBC do MySql para Windows.

CONFIGURAÇÃO:
Criar uma Base com nome BASE_MFB  e importar o script (SCRIPT_BASE_MFB.sql).
No ODBC do windows criar uma fonte de dados de usuario apontando para Banco de Dados BASE_MFB do MySql.
